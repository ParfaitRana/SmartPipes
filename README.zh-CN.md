# Smart Pipes

适用于《星露谷物语》1.6.14+ 与 SMAPI 的自动化物流 Mod。它通过可配置的管道、智能端口和安全出货阀，在箱子、原版生产机器与出货箱之间定向传送物品。

当前正式版：**1.0.0**。

## 安装

1. 安装与游戏版本匹配的 [SMAPI](https://smapi.io/)。
2. 下载并解压 `SmartPipes-1.0.0.zip`。
3. 将其中的 `SmartPipes` 文件夹完整放入游戏的 `Mods` 文件夹。
4. 使用 SMAPI 启动游戏。

可选安装 [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098)：游戏内可在“设置 → Mod Options → Smart Pipes”调整传输频率、吞吐量、性能预算、提示与显示选项。

## 开始使用

达到耕种 4 级且拥有受支持加工机器后，罗宾会寄信提示蓝图。前往木匠店花费 2,000g 购买“基础农场物流蓝图”，即可解锁管道、智能端口和管道扳手配方。

- 对箱子或机器使用智能端口以安装端口。
- 将一格管道铺在端口对象相邻的位置，多个管道连成网络。
- 箱子默认输出，机器默认双向；用扳手交互可调整模式与过滤规则。
- 安全出货阀可安装在主出货箱或迷你出货箱上，用规则控制自动出货。

完整规则说明、机器适配范围、调试命令与测试布局请参阅 [README.md](README.md)。

## 已知限制

- 仅由多人游戏主机执行传输，确保库存不会被多客户端同时修改。
- Mod 不改变原版机器的配方、燃料消耗或加工时间。
- Cask 只会在自然完成后取出成品，不会提前收取银星或金星阶段的酒。

## 升级

直接用新版 `SmartPipes` 文件夹替换旧版本即可。已有端口、阀门和规则会保留；建议在替换前备份存档。
