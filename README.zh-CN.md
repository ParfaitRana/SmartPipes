# Smart Pipes（智能管道）

Smart Pipes 是《星露谷物语》的物流自动化 Mod。你可以用管道连接箱子、原版机器与出货箱，再用端口的方向、过滤和优先级规则决定每件物品应当去往哪里。

它遵循一条核心原则：**只有存在合法目标时，物品才会移动。** 机器或箱子已满时，原料会留在原处，绝不会因为没有地方放而被偷偷送进出货箱。

当前正式版：[1.0.0](https://github.com/ParfaitRana/SmartPipes/releases/latest)

## 安装

1. 安装 [SMAPI](https://smapi.io/)。
2. 从 [最新 Release](https://github.com/ParfaitRana/SmartPipes/releases/latest) 下载 `SmartPipes-1.0.0.zip`。请不要点击 GitHub 仓库页面的 `Code → Download ZIP`，那是源码而不是 Mod 安装包。
3. 解压下载的 ZIP，得到 `SmartPipes` 文件夹。
4. 将整个 `SmartPipes` 文件夹拖入《星露谷物语》的 `Mods` 文件夹。
5. 用 SMAPI 启动游戏。

需要《星露谷物语》1.6.14+ 与 SMAPI 4.2.0+。

## 如何解锁

当你的耕种等级达到 4 级，并且拥有一台受支持的加工机器后，罗宾会在次日早晨寄信。随后可前往木匠店，以 2,000g 购买 **基础农场物流蓝图**。

购买蓝图会解锁以下配方：

- **智能物品管道**：铺设物流网络。
- **智能端口**：安装在箱子或机器上，控制物品流动。
- **管道扳手**：配置、拆除端口与阀门。

购买时还会赠送一把扳手。完成罗宾的入门管网任务，可以获得 10 段管道和 4 个智能端口；次日刘易斯会寄来 **安全出货阀** 配方。

旧存档中已经学会的配方会保留。

## 搭建第一条管线

最基础的布局是：原料箱自动给机器供料，机器完成后将成品送入成品箱。

1. 放置一个**原料箱**、一台受支持机器和一个**成品箱**。
2. 铺设管道，使每个对象都紧贴至少一格管道，且所有管道相互连通。
3. 手持智能端口，对每个箱子和机器按互动键安装端口。
4. 原料箱保持 **输出**，机器设为 **双向**，成品箱设为 **输入**。
5. 将有效原料放进原料箱。

机器能接受原料时会自动开始工作；完成后，只要成品箱有空间，成品就会自动移入其中。

管道属于地面基础设施，玩家和 NPC 都可以直接走过。默认情况下，未手持 Smart Pipes 相关物品时管道会隐藏，不会破坏农场外观。

## 智能端口

智能端口直接安装在箱子、机器或受支持储物设备上，不额外占用地图格。

### 流动模式

| 模式 | 含义 |
| --- | --- |
| **输出** | 允许管网从该对象取出物品。适合原料箱和被动产出设备。 |
| **输入** | 允许管网向该对象放入物品。适合成品箱。 |
| **双向** | 允许取出与放入。大多数加工机器建议使用此模式。 |
| **停用** | 禁止该端口的一切传输。 |

新箱子通常默认输出；大多数受支持机器默认双向。蜂房、树液采集器等只产出物品的设备默认输出。

### 配置与拆除

手持管道扳手时：

- **左键点击**已安装的智能端口或安全出货阀，打开配置界面；
- 对其按**互动键**，依次切换流动模式；
- 按住 **Shift** 再按互动键，安全拆除并返还物品。

用正常工具敲掉带附件的对象时，智能端口或安全出货阀也会单独掉落。因此移动箱子时不会损失附件。

## 过滤、优先级与库存规则

扳手配置界面可以精确控制每一个端口。

### 白名单与黑名单

在过滤输入框中输入物品名称或 ID，也可以从快捷栏/背包中选择物品，然后点选 **Allow（允许）** 或 **Block（拦截）**。

- 白名单为空时，所有未被拦截的物品都可以通过。
- 白名单有内容时，只有名单内物品可以通过。
- 黑名单永远优先于白名单。

例如：让成品箱只允许 Wine（酒）进入。葡萄、水果等未使用原料就会留在原料箱中，而不会误流入成品箱。

### 星级范围

通过星级滑杆的两个手柄选择可接受范围：普通、银星、金星至铱星。它尤其适合分拣木桶陈酿品，或只出售铱星产品。

示例：将目标箱或出货阀设为 **铱星 – 铱星**，并允许 Wine，则只有铱星酒可以进入。

### 优先级

同一件物品有多个合法目标时，优先级更高的端口会先被选择。优先级相同的机器会轮流获得原料，因此一排相同机器不会总是从第一台开始填满。

### 保留数量与库存上限

- **Keep（保留）**：在输出容器中至少留下指定数量。例如设为 20 后，管网不会取走最后 20 件符合条件的物品。
- **Stock cap（库存上限）**：限制输入容器可接收的匹配物品数量。适合避免一个成品箱吸走全部产物。

## 安全出货阀

安全出货阀是 Smart Pipes 唯一可以自动向出货箱送货的方式。它兼容农场主出货箱和迷你出货箱。

1. 手持安全出货阀，对出货箱按互动键安装。
2. 在出货箱占用范围旁边放置至少一格管道。
3. 手持管道扳手，左键点击出货箱打开阀门配置。
4. 设置白名单/黑名单与星级范围。
5. 将模式从 **停用** 切换为 **输入**。

阀门初始为停用，这是刻意的安全设计。启用前请先配置规则。若启用后白名单仍为空，阀门会接受所有未被黑名单拦截的物品，因此强烈建议添加明确的白名单。

主出货箱被罗宾移动时，阀门和其规则会随箱子保留。迷你出货箱仍遵守原版格数上限，装满后管网会安全停止传入。

## 支持的对象

Smart Pipes 支持普通箱子、大箱子，以及可作为来源的自动抓取器（Auto-Grabber）。同时支持以下原版设备：

- 酒桶、罐头瓶、奶酪压制机、蛋黄酱机、织布机、产油机和木桶；
- 脱水机；
- 熔炉、重型熔炉和熏鱼机；
- 树液采集器与重型树液采集器；
- 蜂房；
- 宝石复制机；
- 蘑菇树桩与蘑菇盒子。

Mod 使用原版机器自身的规则，不改变配方、燃料消耗、加工时间或木桶陈酿时间。尤其是：木桶内物品只有自然完成后才会被取出，Mod 不会提前收取银星或金星阶段的陈酿品。

## 设置

安装了 [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) 后，可进入：

**设置 → Mod Options → Smart Pipes**

在这里可以调整传输频率、单次吞吐、大型管网性能预算、世界显示和提示消息。不安装该 Mod 也能正常游玩；SMAPI 会在 Smart Pipes 文件夹中生成 `config.json`。

常用显示设置：

- **Hide pipes when not building**：未手持 Smart Pipes 物品时隐藏管道。
- **Show installed attachments without wrench**：未手持扳手时是否显示已安装端口和阀门；流向箭头始终只在手持扳手时显示。
- **Show transfer messages** 与 **Show installation messages**：可关闭以减少提示文字。

## 多人与大型农场

多人游戏中，只有主机实际改变库存，避免重复传输并保持所有玩家同步。

即使玩家不在建筑内，管网仍会在农场建筑和其他已保存地点运行。面对大型农场，Smart Pipes 会缓存管网结构、让机器公平轮换，并限制单次处理时间以避免卡顿。

## 常见问题

### 为什么物品完全不动？

按顺序检查：

1. 每个端点都必须紧贴一格已连通的管道；
2. 来源必须是输出或双向，目标必须是输入或双向；
3. 物品需要同时通过来源与目标的过滤和星级范围；
4. 目标必须有空间；机器还必须处于可接受原料的状态；
5. 端口或阀门不能处于停用状态。

### 为什么原料没有进入机器？

机器可能需要完整批量或燃料。例如熔炉需要足够矿石加煤炭，脱水机需要完整一组原料。只有原版机器确认可以开工后，Smart Pipes 才会从来源扣除物品；条件不足时原料会安全留在箱中。

### 为什么完成品还留在机器里？

下游箱子可能已满、被停用，或其过滤规则拒绝了该物品。完成品会安全留在机器中，直到出现合法目的地。

### 为什么看不见管道、端口或箭头？

手持任意 Smart Pipes 物品可显示隐藏管道；手持管道扳手可显示流向箭头。端口与阀门的显示规则可在 Mod Options 中调整。

### 如何移动带附件的箱子或出货箱？

可以先按住 Shift、手持扳手并按互动键拆下附件；也可以按正常方式用工具敲掉对象，附件会单独返还。

## 更新

下载新版 Release，解压后用新的 `SmartPipes` 文件夹替换 `Mods` 中的旧文件夹即可。已铺设管道、端口、阀门和规则会保留；每次更新 Mod 前仍建议备份存档。
