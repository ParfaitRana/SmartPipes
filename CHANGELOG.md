# Changelog

All notable changes to Smart Pipes are documented here.

## 1.0.0 - 2026-07-23

First stable release.

- Added directional Smart Pipes with passable, optional hidden world visuals.
- Added Smart Ports with input/output modes, priority, reserve, stock cap, allow/deny lists, and quality-range filtering.
- Added Safe Shipping Valves for the main and Mini-Shipping Bins.
- Added Pipe Wrench installation, removal, interaction, and in-game configuration menus.
- Added support for chests, Big Chests, Auto-Grabbers, artisan machines, Casks, Dehydrators, Furnaces, Fish Smokers, Tappers, Bee Houses, Crystalariums, Mushroom Logs, and Mushroom Boxes.
- Added cached, fair, time-budgeted large-network scheduling across saved locations.
- Added Generic Mod Config Menu integration and a Robin/Lewis unlock story.
- Added English and Simplified Chinese localization.

## Pre-1.0 development

Development builds from 0.8 through 0.11 established the feature set released in 1.0.0.
